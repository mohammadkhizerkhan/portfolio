MINION TRANSLATOR APP

june 07,2021

This is a translator app where it simply translates english to minions language,
We will build this project with help of pure vanilla JS.

live project view in github

Breaking-down the project

  • a)Input

    1)Taking input from user from textarea tag

    • We can also take input from input tag or other ways,
    • After that we have to select the textarea tag through ID by querySelector.
      var txtIn = document.querySelector("#userinput");
    • After that We need to read the contents or the values in the textarea.
      var inputTxt=txtIn.value

    2)Adding button

    • We can add the button by button tag or by input tag where you just have to give attribute type button.
    • For this app we are using button tag with ID="clickme"
      "<"button id="clickme" class="btn">Translate"<"/button>
    • After adding button we have to select the tag and add a event listener "click" to it,So that every time button is clicked the event function should run.
    • We can implement the above point in JS file
      var btn = document.querySelector("#clickme");
      btn.addEventListener("click", function function_name(){ })

  • b)Processing

    1)Taking the serverURL

    • We will store the server url in the variable.
      var serverURL = "https://api.funtranslations.com/translate/minion.json";
    • We will write a function where it appends the text which came from input to the serverURL and returns the final serverURL which need to be requested.

    2)Requesting the server

    • We talked about adding event to button in input section,where we added function which will run everytime a button is clicked,we will request the server through this function(btnHandler()).
    • First we will fetch the serverURL.
    • After fetching will convert response into json.
    • After that we will return the value of request.
      It can be done by specifing the translated text which is present in the contents.translated.
    • we can call this API 5 times per hour,because it is free.After that it will not show output so we have to call an error when server is down.
    • We can call the error by errorHandler function where it will throw an error whenever server is down.

  • c)output

    • First we need to create output space (textarea).
    • After that we have to select the output tag through ID by querySelector
      var output = document.querySelector("#output");
    • With the help of innerText we can display our output in the output textarea section,where we already written inside the btnHandler() function.
